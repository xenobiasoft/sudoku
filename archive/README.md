# Archive

This folder contains projects that have been retired and removed from the active solution.

## Sudoku.Blazor

Archived: 2026-05-10

The Blazor Server frontend was the original UI for this application. Per [ADR-007](../docs/adr/ADR-007-react-vite-strategic-ui.md), React/Vite was designated as the strategic long-term UI target. Once React achieved feature parity, Blazor was retired and the primary custom domain (`sudoku.xenobiasoft.com`) was reassigned to the React Static Web App.

The source is preserved here for reference. It is no longer built, tested, or deployed.
